# Client-Server-File-Transfer

โปรแกรมนี้เป็นระบบ **รับ–ส่งไฟล์ผ่านเครือข่าย (Client–Server)** เขียนด้วยภาษา Java โดยใช้ Socket (NIO) สำหรับการสื่อสารระหว่างเครื่อง Client และ Server

---

## ภาพรวมการทำงาน

- **Server**
  - เปิดรอการเชื่อมต่อจาก Client
  - เมื่อ Client เชื่อมต่อ จะทำการส่งไฟล์ที่กำหนดไปให้ Client
  - รองรับการเชื่อมต่อหลายครั้งแบบต่อเนื่อง
  - แสดงสถานะการเชื่อมต่อและการส่งไฟล์

- **Client**
  - เชื่อมต่อไปยัง Server ตาม IP และ Port ที่กำหนด
  - รับไฟล์จาก Server และบันทึกลงเครื่อง
  - แสดงความคืบหน้าในการรับไฟล์
  - จัดการกรณี timeout หรือการเชื่อมต่อขาดหาย

---
## อธิบายหน้าที่ของแต่ละไฟล์

### 1. `ServerMain.java`
- เป็น **จุดเริ่มต้นของฝั่ง Server**
- ทำหน้าที่:
  - สร้าง `ServerSocketChannel`
  - กำหนดพอร์ตสำหรับรอรับการเชื่อมต่อ
  - รอ Client เชื่อมต่อเข้ามา
  - เรียกใช้งานคลาส `Server` เพื่อจัดการการส่งไฟล์

---

### 2. `Server.java`
- ทำหน้าที่หลักในการ **ส่งไฟล์ให้ Client**
- ความสามารถหลัก:
  - เปิดไฟล์จากเครื่อง Server
  - อ่านไฟล์เป็น buffer แล้วส่งผ่าน `SocketChannel`
  - ตรวจสอบสถานะการเชื่อมต่อของ Client
  - แสดงข้อความเมื่อ Client ตัดการเชื่อมต่อ

---

### 3. `ClientMain.java`
- เป็น **จุดเริ่มต้นของฝั่ง Client**
- ทำหน้าที่:
  - กำหนด IP และ Port ของ Server
  - สร้างการเชื่อมต่อไปยัง Server
  - เรียกใช้งานคลาส `Client` เพื่อรับไฟล์

---

### 4. `Client.java`
- ทำหน้าที่หลักในการ **รับไฟล์จาก Server**
- ความสามารถหลัก:
  - รับข้อมูลไฟล์ผ่าน `SocketChannel`
  - เขียนไฟล์ลงเครื่องด้วย `FileOutputStream`
  - แสดง progress การดาวน์โหลดไฟล์
  - ตรวจสอบ timeout และจัดการกรณีเชื่อมต่อผิดพลาด

---

## ลำดับการทำงานโดยสรุป

1. รัน `ServerMain`
2. Server เปิดพอร์ตและรอ Client
3. รัน `ClientMain`
4. Client เชื่อมต่อไปยัง Server
5. Server ส่งไฟล์ให้ Client เป็นชุดข้อมูล (buffer)
6. Client รับข้อมูลและเขียนเป็นไฟล์
7. เมื่อส่งเสร็จ ทั้งสองฝั่งปิดการเชื่อมต่อ
